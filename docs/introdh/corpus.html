Corpus Analysis

A corpus analysis on Star Wars!

For my corpus analysis I chose the scripts of the Star Wars movies. I knew that each movie had a different director and writer, so I wanted to see if I could find any patterns in the scripts. I am going to break up the corpuses into the first 4 movies and the last 3 movies. I will then compare the corpuses to see if there are any differences in the writing styles of the different directors and writers.

What scripts did I use for the corpus and how did I obtain them?

I am using the scripts from the first 7 Star Wars movies. I obtained them from The Internet Movie Script Database (IMSDb). I copied and pasted the scripts into a text file and then saved them as a .txt file.

What I found interesting about the corpus and what I learned from it?

Having a look at each ngram at size of three words, I found that the first 4 movies focuses on Luke way more than any other character. The last 3 movies kinda focuses on other things.
